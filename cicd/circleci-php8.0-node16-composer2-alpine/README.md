# silta-circleci
An Alpine based CircleCI builder image with the following additions:

- Composer
- Drush-launcher and coder pre-installed
- Vim
- Google cloud CLI, Azure CLI, Kubernetes CLI, Helm

Note: Alpine Linux uses musl.

## Versions
- PHP: 8.0.13
- Composer: 2.1.12
- Node: 16.13.0
- GCloud: 348.0.0-0
- Helm: v3.6.3
