# silta-circleci
A docker image used circleCI, based on `cimg/php:8.3.12-node` with the following additions:

- Composer configured correctly
- Drush-launcher and coder pre-installed
- Vim, useful for debugging
- kubernetes and helm

## Versions
- PHP: 8.3.12
- Composer: 2.5.1
- Node: 20.11.1
- Helm: v3.14.0
