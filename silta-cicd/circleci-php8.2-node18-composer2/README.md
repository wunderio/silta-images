# silta-circleci
A docker image used circleCI, based on `cimg/php:8.2.12` with the following additions:

- Composer configured correctly
- Drush-launcher and coder pre-installed
- Vim, useful for debugging
- kubernetes and helm
- Node.js
- Yarn

## Versions
- PHP: 8.2.0
- Composer: 2.1.12
- Node: 18.18.2
- Yarn: 1.22.19
- Helm: v3.14.0
