# silta-circleci
A docker image used circleCI, based on `cimg/php:8.0.13` with the following additions:

- Composer configured correctly
- Drush-launcher and coder pre-installed
- Vim, useful for debugging
- kubernetes and helm
- Node.js
- Yarn

## Versions
- PHP: 8.0.13
- Composer: 2.1.12
- Node: 16.20.2
- Yarn: 1.22.5
- Helm: v3.16.3
