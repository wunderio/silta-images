# silta-circleci
A docker image used circleCI, based on `cimg/php:8.0.13-node` with the following additions:

- Composer configured correctly
- Drush-launcher and coder pre-installed
- Vim, useful for debugging
- kubernetes and helm

## Versions
- PHP: 8.0.13
- Composer: 2.1.12
- Node: 16.13.0
- Helm: v3.13.3
