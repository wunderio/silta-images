# silta-circleci
A docker image used circleCI, based on `cimg/php:8.2.20` with the following additions:

- Composer configured correctly
- Drush-launcher and coder pre-installed
- Vim, useful for debugging
- kubernetes and helm
- Node.js
- Yarn

## Versions
- PHP: 8.2.20
- Composer: 2.5.1
- Node: 22.3.0
- Yarn: 1.22.22
- Helm: v3.14.0
