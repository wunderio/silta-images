# silta-circleci
A docker image used circleCI, based on `cimg/php:8.1.23` with the following additions:

- Composer configured correctly
- Drush-launcher and coder pre-installed
- Vim, useful for debugging
- kubernetes and helm
- Node.js
- Yarn

## Versions
- PHP: 8.1.23
- Composer: 2.5.1
- Node: 18.17.1
- Yarn: 1.22.19
- Helm: v3.14.0
