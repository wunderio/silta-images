# silta-circleci
A docker image used circleCI, based on `cimg/php:8.1.7` with the following additions:

- Composer configured correctly
- Drush-launcher and coder pre-installed
- Vim, useful for debugging
- kubernetes and helm
- Node.js
- Yarn

## Versions
- PHP: 8.1.7
- Composer: 2.1.12
- Node: 16.15.1
- Yarn: 1.22.5
- Helm: v3.14.0
